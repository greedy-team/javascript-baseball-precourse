## 기능목록

1. random 수 생성
    1. 1~9까지 서로 다른 수

2. user input 검사
    1. 잘못된 값 alert로 메시지
    2. 길이가 3인지 
    3. 서로 다른 세가지 수인지
    4. 1~9 사이의 수인지

3. usernum comnum 비교
    1. 위치 숫자 모두 같으면 strike
    2. 숫자만 같고 위치가 틀리면 ball

4. ball과 strike 결과 출력
    1.ball과 strike 둘다 있다면 ball 먼저 출력

5. 게임 종료 후 재시작 버튼 누르면 reset
